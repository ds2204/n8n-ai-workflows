Get Your AI Agents Running

Step-by-step guide to deploying all 7 workflows with Docker, credentials, and n8n configuration.

Prerequisites

Before starting, make sure you have the following installed on your machine:

REQUIRED
Docker Desktop
Install from docker.com. Required to run n8n in a container.
REQUIRED
API Keys
OpenAI (GPT-4o-mini), Groq (LLaMA 3), and any other services you want to use.
OPTIONAL
Pinecone Account
Only needed for workflow #04 (RAG Agent). Free tier is sufficient.
OPTIONAL
Google Cloud Project
For Gmail and Google Drive OAuth2. Needed for workflows #03 and #04.

Spin Up n8n with Docker

Navigate to the folder containing docker-compose.yml and run:

# 1. Copy the env template
cp .env.example .env

# 2. Edit .env with your API keys
nano .env   # or open in your editor

# 3. Start n8n
docker compose up -d

# 4. Check it's running
docker compose ps

# 5. View logs (optional)
docker compose logs -f n8n

Once running, open http://localhost:5678 in your browser. Log in with the username/password from your .env file (admin / changeme123 by default).

⚠ Change your password! Before exposing n8n to the internet, update N8N_BASIC_AUTH_PASSWORD in your .env file to something secure.

Import Workflows

All workflow JSON files are in the workflows/ folder. Import them into n8n:

Option A — Via UI (Easy)

1. Open http://localhost:5678
2. Click "Workflows" in the left sidebar
3. Click "Import from file" (top right)
4. Select each .json file from the workflows/ folder
5. Save each workflow after importing

Option B — Via CLI

# Import all workflows at once
docker exec -it n8n-ai-agent-lab n8n import:workflow --separate --input=/home/node/.n8n/workflows

You should now see all 7 workflows listed in your n8n dashboard.

Getting Your OpenAI Key

1. Go to platform.openai.com
2. Settings → API Keys → Create new secret key
3. Copy and paste into n8n credential

Getting Your Groq Key

1. Go to console.groq.com
2. API Keys → Create API Key
3. Copy and paste into n8n credential

Setting Up Gmail OAuth2

1. Go to console.cloud.google.com
2. Create a new project (or use existing)
3. Enable "Gmail API" and "Google Drive API"
4. OAuth consent screen → External → Add your email as test user
5. Credentials → Create OAuth 2.0 Client ID
   - Application type: Web application
   - Authorized redirect URI: http://localhost:5678/rest/oauth2-credential/callback
6. Copy Client ID and Secret into n8n
7. Click "Sign in with Google" in n8n to authorize

Setting Up Pinecone

1. Go to pinecone.io → Sign up (free tier works)
2. Create a new index:
   - Name: ai-knowledge-base
   - Dimensions: 1536  (for text-embedding-3-small)
   - Metric: cosine
3. API Keys → Copy your key
4. Paste into n8n Pinecone credential

Activate & Test Each Workflow

Workflow #01 — Feedback Classifier

1. Open workflow in n8n
2. Toggle "Active" switch ON
3. Click "Form Trigger" node → copy the form URL
4. Open the URL in your browser
5. Submit test feedback
6. Check execution log — should show "complaint", "appreciation", or "feature_request"

Workflow #02 — Webhook Chatbot

1. Toggle "Active" ON
2. Copy webhook URL from the Webhook node
3. Test with curl:
   curl -X POST http://localhost:5678/webhook/chat \
     -H "Content-Type: application/json" \
     -d '{"message": "Hello, what can you do?"}'
4. You should receive a JSON response with "reply" field

Workflow #03 — Email Reply Agent

1. Set up Gmail OAuth2 credential first
2. Toggle "Active" ON
3. The agent will run every 5 hours automatically
4. To test immediately: click "Execute Workflow" button
5. Check execution log for processed emails

Workflow #04 — RAG Pipeline

1. Set up Pinecone + OpenAI + Google Drive credentials
2. Import 04a (Builder) first, set your Google Drive folder ID
3. Toggle 04a "Active" ON — it will auto-process new PDFs
4. Import 04b (Chatbot) and toggle "Active" ON
5. Open the chat interface URL from the Chat Trigger node
6. Ask questions about your documents

Workflow #06 — Twitter Trend Agent

1. Set up Twitter OAuth2 credential
2. Toggle "Active" ON
3. Open chat interface URL
4. Ask: "What are people saying about AI agents on Twitter?"
💡 Pro Tip: Use n8n's built-in "Test workflow" feature (play button) before activating any workflow. This runs a single execution you can debug step-by-step without scheduling or webhooks firing.

Troubleshooting

n8n won't start

# Check if port 5678 is already in use
lsof -i :5678

# Check Docker logs
docker compose logs n8n

# Restart fresh
docker compose down && docker compose up -d

Credentials not working

# Common issues:
- OpenAI: Make sure you have billing enabled at platform.openai.com
- Gmail: OAuth redirect URI must exactly match: 
  http://localhost:5678/rest/oauth2-credential/callback
- Pinecone: Index dimensions must be 1536 for text-embedding-3-small
- Groq: Free tier has rate limits — add delays if needed

AI Agent not responding

# Check that:
1. The language model node is connected to the agent
2. Your API key has credits/quota remaining
3. The "ai_languageModel" connection type is correct
4. Check n8n execution logs for the exact error message
